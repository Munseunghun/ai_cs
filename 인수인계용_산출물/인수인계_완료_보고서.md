## 🎯 주요 기능

### 1. 대시보드
- 실시간 통계 카드 (진행 중/예정/채널 수)

### 2. 이벤트 검색

- 정렬 (최신순, 인기순, 종료임박순)
- 페이지네이션 (20개씩)

### 3. 라이브 상세 조회
- 기본 정보 (제목, 브랜드, 채널, 일시)
- 상품 정보 (18개 평균, 중복 제거)

- CS 응대 정보 (FAQ, 응답 스크립트, 리스크 포인트)
- 고객 반응 (댓글, 좋아요, 조회수)
- 통계 정보 (시청자 수, 판매 수량)

- 상세 로그 기록

## 🚀 빠른 시작 가이드

### 로컬 개발 환경 구축

**1. 저장소 클론**:
```bash
git clone https://github.com/Munseunghun/ai_cs.git
cd ai_cs
```

**2. 프론트엔드 실행**:
```bash
cd frontend
npm install
npm start
# http://localhost:3000
```

**3. 백엔드 실행**:
```bash
cd backend
npm install
npm run dev
# http://localhost:3001
```

**4. 크롤러 실행**:
```bash
cd crawler
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 comprehensive_naver_crawler.py
```
