## 🐛 트러블슈팅

### 문제 1: 버튼이 여전히 표시됨

**원인**: 브라우저 캐시

**해결**:
```bash
# 브라우저 캐시 삭제
Ctrl + Shift + Delete (Windows)
Cmd + Shift + Delete (Mac)

# 또는 하드 리프레시
Ctrl + F5 (Windows)
Cmd + Shift + R (Mac)
```

### 문제 2: 레이아웃이 깨짐

**원인**: CSS 충돌

**해결**:
```javascript
// Box 컴포넌트의 sx 속성 확인
<Box sx={{ 
  textAlign: 'center',  // 중앙 정렬
  mt: 4,                // 상단 마진
  display: 'flex',      // Flexbox 사용 (선택)
  justifyContent: 'center',  // 중앙 정렬 (선택)
}}>
```

### 문제 3: 버튼 클릭이 작동하지 않음

**원인**: navigate 함수 오류

**해결**:
```javascript
// useNavigate 훅 확인
import { useNavigate } from 'react-router-dom';

const navigate = useNavigate();

// 버튼 클릭 핸들러 확인
onClick={() => navigate('/search')}
```
