## 🎯 주요 기능

### 1. 대시보드
- 실시간 통계 (진행 중/예정/채널 수)

- 페이지네이션

### 3. 라이브 상세 조회
- 기본 정보 (제목, 브랜드, 채널)
- 상품 정보 (중복 제거)
- 혜택 정보 (쿠폰, 사은품)
- CS 응대 정보 (FAQ, 응답 스크립트)
- 고객 반응 (댓글, 조회수)
