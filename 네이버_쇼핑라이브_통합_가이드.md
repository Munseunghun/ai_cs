## 개요

네이버 쇼핑라이브([https://view.shoppinglive.naver.com](https://view.shoppinglive.naver.com))의 라이브 방송 정보를 크롤링하여 CS 상담원이 효율적으로 고객 응대를 할 수 있도록 지원하는 통합 시스템입니다.

### ✨ 핵심 기능

- **멀티 브랜드 크롤링**: 10개 주요 브랜드 자동 수집
- **상세 정보 수집**: 방송별 8개 카테고리, 50+ 항목 수집
- **통합 검색**: 브랜드, 상품명, 키워드 기반 검색
- **CS 응대 지원**: 예상 질문, 응답 스크립트, 리스크 포인트 제공

## 상세 조회 시스템

### 상세 조회 화면 구성

상세보기 페이지(`/live/:liveId`)에서 다음 8개 섹션이 표시됩니다:

#### 1) 기본 정보
- 라이브 ID, 플랫폼명, 브랜드
- 라이브명(고객용/CS용)
- 원천 URL

#### 2) 방송 스케줄 & 혜택 유효시간
- 방송일자, 방송시간
- 혜택 유효기간 타입 및 일시
- 방송 형태

#### 3) 판매 상품 정보
- 상품명, SKU, 가격, 할인율
- 재고 정보, 세트 구성
- 상품별 상세 정보 테이블

#### 4) 혜택(Promotion) 구조
- **할인**: 할인 유형, 상세, 조건
- **사은품**: 사은품 유형, 조건, 수량 제한
- **쿠폰/적립**: 쿠폰 유형, 발급/사용 조건
- **배송**: 배송 혜택, 조건

#### 5) 중복 적용 정책 (Risk Point)
- 쿠폰 중복 가능 여부
- 적립 중복 가능 여부
- 타 행사 중복 정책
- 임직원 할인 적용 여부

#### 6) 제외/제한 사항
- 제외 상품 목록
- 채널 제한
- 결제수단 제한
- 지역/배송 제한

#### 7) 라이브 특화 정보 (STT 기반)

## 문제 해결

### 크롤링 관련

#### ChromeDriver 에러
```
selenium.common.exceptions.WebDriverException
```
**해결 방법**:
```bash
# ChromeDriver 설치 확인
brew install chromedriver
chromedriver --version

# 경로 확인
which chromedriver
```

#### 페이지 로드 실패
**해결 방법**:
- 네트워크 연결 확인
- 네이버 쇼핑라이브 URL 유효성 확인
- 대기 시간 증가 (time.sleep)

#### 데이터 파싱 실패
**해결 방법**:
- 네이버 쇼핑라이브 페이지 구조 변경 가능성 확인
- HTML 선택자 업데이트 필요
- 파서 코드 수정

### 프론트엔드 관련

#### 데이터가 조회되지 않는 경우

1. **Mock 데이터 확인**:
```javascript
// frontend/src/mockData/allBrandsLiveData.js
export const allBrandsLiveData = [ ... ];
```

2. **검색어 확인**:
   - 브랜드명 정확히 입력 (예: "라네즈" O, "라네쥬" X)
   - 키워드 대소문자 구분 없음

3. **브라우저 콘솔 확인**:
   - F12 → Console → 에러 메시지 확인

#### 상세보기 페이지가 열리지 않는 경우

1. **라우팅 확인**:
   - `frontend/src/App.jsx`에서 `/live/:liveId` 경로 확인

2. **liveId 확인**:
   - 올바른 liveId 형식인지 확인
   - Mock 데이터에 해당 liveId가 있는지 확인

### 성능 관련

#### 크롤링이 너무 느린 경우

1. **대기 시간 조정**:
```python
# 서버 부하 방지 대기 시간 (초)
time.sleep(2)  # 기본 2초
time.sleep(1)  # 더 빠르게 (권장하지 않음)
```

2. **수집 개수 제한**:
```python
# 브랜드당 최대 3개만 수집
_v_all_lives = _v_crawler.crawl_all_brands(p_max_per_brand=3)
```

#### 프론트엔드가 느린 경우

1. **데이터 최적화**:
   - Mock 데이터 크기 확인
   - 불필요한 데이터 제거

2. **페이지네이션 확인**:
   - 검색 결과 페이지네이션 적용 여부 확인
