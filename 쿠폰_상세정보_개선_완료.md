## 🔄 데이터 매핑

### 지원하는 필드
```javascript
// 쿠폰 기본 정보
- benefit_name / coupon_name          // 쿠폰 이름
- benefit_type / coupon_type          // 쿠폰 타입
- benefit_detail / coupon_description // 쿠폰 설명
- coupon_discount_value               // 할인 금액
- coupon_discount_type                // 할인 타입

// 조건 정보
- benefit_condition / min_purchase_amount  // 최소 구매금액
- max_discount_amount                      // 최대 할인금액
- coupon_issue_limit                       // 발급 제한
- duplicate_use                            // 중복 사용 가능 여부

// 기간 정보
- benefit_valid_period                     // 유효기간 (통합)
- coupon_valid_start                       // 유효기간 시작
- coupon_valid_end                         // 유효기간 종료

// 추가 정보
- coupon_issue_condition                   // 발급 조건
- target_products                          // 적용 상품
- excluded_products                        // 제외 상품
- coupon_code                              // 쿠폰 코드
- usage_count                              // 사용 횟수
- coupon_status                            // 쿠폰 상태
```
